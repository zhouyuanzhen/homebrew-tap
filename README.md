# homebrew-tap

Yuanzhen 's HomeBrew Tap repository

## QuickStart

### Tap This Repo

```shell
# (Setup this tap)
brew tap zhouyuanzhen/tap

brew update
```

### Play with homebrew

```shell

brew install zhouyuanzhen/tap/ahyve
ahyve -h

brew install --HEAD zhouyuanzhen/tap/helloworld-c
# OR
brew install --HEAD helloworld-c

```
